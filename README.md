## 簡介
這是一個使用**Discord.py**與**Gemini API**開發的Discord bot    
* 功能
    * 在一般頻道mention你的機器人或者reply機器人的任意訊息，即可與AI對話
    * 可以讀取訊息中附帶的圖片
    * 支援語音功能，可與使用者在語音頻道中對話
    * 在`settings/system_prompt`中可以寫系統提示詞，自訂你的AI
